# Rotina Diária AI

Plugin WordPress em PHP para automatizar a criação de conteúdo focado em SEO.

## Objetivo
Automatizar a rotina diária de criação de posts para blogs monetizados com AdSense,
voltados para público feminino, maternidade, gravidez, significados de nomes e presentes.

## Público-alvo
Mulheres, mães, gestantes e famílias.

## Tecnologias
- WordPress
- PHP
- MySQL
- API de IA (OpenAI ou Gemini)
- Rank Math (FAQ Block)
- GenerateBlocks

## Funcionalidades principais

### 1. Painel no WordPress
Plugin com painel próprio no admin, com abas horizontais.

Abas:
- Configurações
- Post SEO
- Melhor dos Melhores
- Agentes Especialistas

### 2. Configurações
- Campo para API Key
- Futuro: múltiplas chaves
- Salvamento via options do WordPress

### 3. Post SEO
- Campo para palavra-chave
- Escolha de tamanho:
  - Pequeno (até 1500 palavras)
  - Médio (até 2500 palavras)
  - Grande (até 5000 palavras)
- Geração em etapas:
  1. Estrutura (H2)
  2. Conteúdo por seção (H3, listas, tabelas)
- Inclusão de FAQ no final
- Envio automático como rascunho ou publicado

### 4. Melhor dos Melhores
- Campo para URL de artigo externo
- Extração apenas do conteúdo principal
- Tradução e reescrita por seções
- Manter estrutura original
- Crédito automático no final:
  "Com conteúdo de [Nome do Site](link)"

### 5. Agentes Especialistas
Agentes de IA focados em:
- Ideias de monetização
- Criação de produtos digitais
- Estratégias de conteúdo
- Mentoria passo a passo

## Regras Importantes
- Código em PHP (WordPress)
- Sem duplicar conteúdo
- Reescrita ética
- SEO-first
- Código organizado e comentado
